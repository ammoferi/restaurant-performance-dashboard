# Restaurant Performance Dashboard

## 📌 Project Overview
This project analyzes restaurant pizza sales data and presents business insights through an interactive Power BI dashboard.

The dashboard helps track:
- Revenue performance
- Order trends
- Product category performance
- Pizza size contribution
- Best and worst selling pizzas

---

## 🛠 Tools Used
- Power BI
- CSV Dataset

---

## 📊 Key Metrics
- Total Revenue: 16,895.40
- Total Orders: 439
- Total Pizzas Sold: 1,022
- Average Order Value: 38.49
- Average Pizzas Per Order: 2.33

---

## 🔍 Key Insights
- Sundays recorded the highest number of orders
- Classic pizzas generated the highest sales
- Large pizzas contributed the largest sales share
- The Thai Chicken Pizza ranked among the best-performing products
- The Calabrese Pizza ranked among the lowest-performing products
